## Excel / Google Sheets Basics
Most data entry jobs require spreadsheets. Let’s start with the very basics.

### A. Spreadsheet Layout
- **Cells** – boxes where data goes (e.g., A1, B2).
- **Rows** – horizontal lines (1, 2, 3…)
- **Columns** – vertical lines (A, B, C…)
- **Sheets** – multiple pages in one workbook.

### B. Basic Actions 

1. **Enter data:** click a cell → type → Enter
2. **Edit data:** double-click cell or type directly
3. **Copy/Paste:** Ctrl + C / Ctrl + V
4. **Auto-fill:** drag the small square at cell corner to fill series
  
### C. Simple Formulas

- **SUM**: Adds numbers

  <code>=SUM(A1:A5)</code>
  
- **AVERAGE**: Finds average

  <code>=AVERAGE(B1:B5)</code>

- **COUNT**: Counts numbers

  <code>=COUNT(C1:C5)</code>
